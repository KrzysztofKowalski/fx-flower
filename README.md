# Geometry calculator

## Requiremens:
```
brew install parallel gnuplot
```
## Usage

#### Generates entire set of 999^3 images

```
./preparecommands.sh `pwd`
./launch.sh
```

## For some nice samples press:

```
./samples.sh
open samples
```
